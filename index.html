🖥️ Local System Diagnostics Script

This script runs on the hotel PC and captures detailed system info that the browser cannot access. It collects everything needed for troubleshooting and copies a WhatsApp-ready report to clipboard.

What it collects:
• Computer hostname, username, domain
• OS version, system uptime, manufacturer/model
• CPU model, cores, clock speed, current load, temperature*
• RAM total/free, DDR type & speed per module
• Disk space, SSD/HDD model
• GPU model and driver version
• Network adapter, link speed, local IP, gateway, DNS
• DNS resolution test for PMS domains
• Ping test (5 packets) to Google, Cloudflare, PMS, API
• Traceroute to PMS server
• Installed browser versions (Chrome, Edge, Firefox)
• Chrome memory usage and tab count
• Top 10 processes by RAM usage
• Antivirus software detected
*Temperature requires running as Administrator
What it does NOT do:
• Does not install anything
• Does not modify system settings
• Does not send data anywhere — report stays on your clipboard and in a local temp file
• Does not require internet (except for ping/traceroute tests)
• Source code is visible — right-click the file and "Edit" to inspect

Download options:

📄 Download as .txt (rename to .bat to run)

Use this if your browser blocks .bat downloads. After downloading, rename sentec-diag.txt → sentec-diag.bat

⚡ Download as .bat (double-click to run)

Direct download — if Chrome blocks it, click the ↑ arrow in the download bar → "Keep"

How to run:
1. Download the file (use .txt version if .bat is blocked)
2. If .txt: rename the file extension from .txt to .bat
3. Double-click sentec-diag.bat to run
4. Wait about 2 minutes for all tests to complete
5. When done, the WhatsApp report is on your clipboard — just paste in the group chat!

💡 Tip: Right-click → "Run as Administrator" to also get CPU temperature readings.

📱 More Devices & Browsers

If the above doesn't work, follow the guide for your specific device:

📱 Chrome — Android▶
  1. Open Chrome → go to pms.sentec.io
  2. Tap ⋮ three dots (top right) → Settings
  3. Tap Privacy and security → Clear browsing data
  4. Set time range to "All time"
  5. Check "Cookies and site data" and "Cached images and files"
  6. Tap "Clear data" → Go back to PMS and log in
💡 Site-specific: On PMS page → tap ⋮ → ⓘ info icon → Site settings → Clear & reset
📱 Chrome — iPhone / iPad▶
  1. Open Chrome app
  2. Tap ⋯ three dots (bottom right) → Settings
  3. Tap Privacy and Security → Clear Browsing Data
  4. Select "All Time"
  5. Check "Cookies, Site Data" and "Cached Images and Files"
  6. Tap "Clear Browsing Data" → Confirm → Log in again
🧭 Safari — iPhone / iPad▶
  1. Open Settings app → scroll down → tap Apps → Safari
  2. Tap "Clear History and Website Data"
  3. Select "All history" → tap "Clear History"
  4. Go back to PMS in Safari and log in again
💡 Site-specific: Settings → Apps → Safari → Advanced → Website Data → search "sentec" → swipe left → Delete
🧭 Safari — Mac▶
  1. Open Safari → Menu bar → Safari → Settings
  2. Go to Privacy tab → click "Manage Website Data"
  3. Search "sentec" → select → click "Remove"
  4. Close Settings → go back to PMS and log in again
